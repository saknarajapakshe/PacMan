# Pac-Man Game

## Installation Instructions

### Option 1: Run JAR File Directly
1. Make sure you have Java installed on your system
2. Double-click `PacMan.jar` or run `java -jar PacMan.jar` from command line

### Option 2: Use Batch Script (Windows)
1. Double-click `RunPacMan.bat` to start the game

### System Requirements
- Java 8 or higher
- Windows, macOS, or Linux
- Sound card for audio effects

### How to Play
- Use arrow keys to move Pac-Man
- Eat all dots and cherries to complete the level
- Avoid ghosts or you'll lose a life
- Press SPACE to pause the game
- Collect cherries for bonus points (100 points each)
- Regular dots are worth 10 points each

### Game Features
- Classic Pac-Man gameplay
- Sound effects
- Cherry collection for bonus points
- Pause functionality
- Tunnel teleportation
- Score tracking
- Lives system

### Troubleshooting
If the game doesn't start:
1. Make sure Java is installed and in your system PATH
2. Try running from command line: `java -jar PacMan.jar`
3. Check that all game assets are in the same directory as the JAR file

Enjoy playing Pac-Man!
